YARP2

  • YARP2 is a new version of YARP.

  • It is undergoing development under modules "src/libYARP_OS2" and "src/java" in the CVS repository.

  • Implementation information for developers: class documentation

  • Specification for YARP version 2: draft protocol document 2.0

  • User guide: there is none yet, but the draft specification describes how the "YARP Companion" program should operate. This program has been implemented partially in the above modules.
